Summary

The purpose of this project was to analyze the Data Job Market (Data Analyts, Data Engineer, Data Scientist) and find some interesting trends. To accomplish this analysis we first pulled our data from Kaggle and cleaned the Data.
After data cleaning, we used leaflet to plot various aspects of Data Job distribution in some of the top cities in United States. Factors we looked at includes: Easy Apply, Job Industry, Salary Range and Job Sector. This site provides the source data and Visualizations created as part of the analysis, as well as explainations and descriptions of any trends witnessed.
